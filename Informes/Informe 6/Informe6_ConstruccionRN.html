Construccin de modelo de Machine Learning supervisado con datos de expresin gnica

En este informe tenemos como objetivo construir un modelo de Machine Learning Supervisado utilizando los datos de Expresin gnica transformados. Adems utilizaremos la versin filtrada de estos datos por un lado, y la versin con nicamente los genes diferencialmente expresados por otro.

Introduccin al aprendizaje supervisado

El aprendizaje supervisado consiste en mapear unos datos de entrada a unas etiquetas targets (tambin conocidas como anotaciones), dados un conjunto de ejemplos anotados por humanos. Casi todas las aplicaciones de Deep Learning actuales pertenecen a esta categora. Por lo que, necesitaremos unas etiquetas (Labels) para nuestros datos, en nuestro caso tomaremos la variable vital_status.

Preparacin datos, conjuntos de test y train y etiquetas

Vamos a cambiar el array y vamos a poner las muestras como filas y las filas como muestras. Vamos a realizar la prueba del modelo de Deep Learning con dos tipos de datos:

  • ExpGenTCGA_KIRC_Norm_Trans_Filt75_238DEG_Transposed: Los datos de Expresion Gnica de KIRC, normalizados, transformados, con filtrado de genes al 75% y con los 238 genes ms diferencialmente expresados. Nmero de muestras = 606, nmero de genes = 238.
  • ExpGenTCGA_KIRC_Norm_Trans_Filt75_Transposed: Los datos de Expresion Gnica de KIRC, normalizados, transformados y con filtrado de genes al 75%. Nmero de muestras = 606, nmero de genes = 4897.

tfruns: Track and Visualize Training Runs

Una mejor manera de comparar los entrenamientos es con el paquete tfruns.

  • Hace un seguimiento de los hiperparmetros, mtricas, resultados y cdigo fuente de cada entrenamiento.
  • Compara hiperparmetros y mtricas de todos los runs y encuentra el mejor modelo.
  • Genera automticamente informes para visualizar indivualmente los entrenamientos o comparacin entre runs.
  • No se necesita cambiar el cdigo fuente.

Instalacin

install.packages("tfruns")
install.packages("tfestimators")

Entrenamiento

En las siguientes secciones se describirn las capacidades de tfruns. Nuestro script de entrenamiento script.R entrena un modelo Keras para clasificar pacientes entre vivos y muertos a partir de datos de Expresin gnica.

Para entrenar la red con tfruns utiliza la funcin training_run() para ejecutar el script de R.

library(tfestimators)
library(tfruns)
training_run("script.R")

Cuando el entrenamiento se haya completado, un resumen de la ejecucin aparecer en pantalla si ests en una sesin interactiva de R.
